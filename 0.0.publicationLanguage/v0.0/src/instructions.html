How to play

Goal:

The winner is the first player who put 4 tokens consecutives:

  • In horizonal (same row)
  • In vertical (same column)
  • In diagonal (+1 row and +1 column each)
  • In diagonal inverse (-1 row and +1 column each)

Game rules:

  • Any player must put one token each turn.
  • Game ends if:
    • A player wins (goal achived).
    • Tie: the board is full.

Put token rules:

A token can be put in any empty square that meets one of the following requirements:
  • Is in the bottom row.
  • The square just below (same column, -1 row) already has a token on it.
